== Description ==

"WP Sales Notifier" does work with [WooCommerce Plugin](https://wordpress.org/plugins/woocommerce/). It's manily created for show to recent sales on the website.

Note : please don't forget to leave your valuable feedback and suggestions for make this plugin much more better compare to other plugins.

== Installation ==

In most cases you can install automatically from WordPress.

However, if you install this manually, follow these steps:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The settings page will be available under settings menu (Settings >> WP Sales Notifier)
